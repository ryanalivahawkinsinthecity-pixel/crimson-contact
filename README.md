# Crimson Contract - Gothic BL Visual Novel

**Description:**  
Crimson Contract is a Gothic BL Visual Novel with vampire themes, branching romance, humanity system, skill tree, combat minigame, cinematic effects (rain, fog, blood flashes), and multiple endings.

**Features:**
- Branching story with multiple endings
- Vampire skill tree
- Combat minigame
- Cinematic effects: screen shake, blood flashes, rain/fog overlays
- Sprites with expressions and blinking
- Music & SFX per scene
- Typewriter-style dialogue
